Help!

Hoi! Goed dat je hier bent. Dit is de FAQ. De afkorting FAQ staat voor Frequently Asked Questions en dat is Engels voor veelgestelde vragen. Antwoorden op deze vragen vind je hier! Mocht je het antwoord op jouw vraag hier niet kunnen vinden, stuur dan een mail naar literamuur@mail.com.

Wat is de LiteraMuur?

De LiteraMuur is een website met literatuur voor de Pabo.

Kan ik ook literatuur insturen?

Jazeker! Dat kan via de LiteraTip pagina.

Waarom is de LiteraMuur open source?

Wij vinden het ontzettend belangrijk dat de LiteraMuur een open en transparant platform is en blijft. Daarom staat de volledige broncode van de website op GitHub en kan iedereen bijdragen aan de broncode. Onze GitHub pagina vind je hier.

Onder welke licentie is deze site gepubliceerd?

De inhoud van de LiteraMuur is vrijgegeven onder de Creative Commons Naamsvermelding-NietCommercieel-GelijkDelen 4.0 Internationale Licentie. De broncode die gebruikt wordt om deze inhoud weer te geven is vrijgegeven onder de GNU GPL v3.
