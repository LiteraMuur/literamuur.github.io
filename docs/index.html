Literatuur

Database
Per boek is de volgende informatie opgenomen in de database:

  • titel
  • schrijver(s)
  • jaar van uitgave
  • beschrijving
  • APA notatie
  • tags (woorden die helpen het boek snel te vinden, bijvoorbeeld de belangrijkste woorden uit het register en deelonderwerpen van het boek)
  • foto van cover
De database bestaat uit een JSON bestand. In de JSON database staat elk boek er als volgt in: 
{
 "id": "00001",
 "titel": "Opbrengstgericht onderwijs ontwerpen",
 "schrijver": "Anouke Bakx, Anje Ros & Peter Teune",
 "beschrijving": "Super interessante beschrijving.",
 "vakgebied": "pedagogiek",
 "tags": ["woorden, die, relevant, kunnen, zijn, gescheiden, door, een, komma"]
}
Het jaartal en de APA omschrijving zijn niet opgenomen in de JSON database. Deze informatie staat alleen op de boekpagina (/boeken/id.html).
